原因以及解法适用

原因

可能是梯子的原因吧 qwq。

解法适用

微软账号登录是显示的错误

解法

关掉梯子，关掉梯子！

打开 cmd (win + r 输入 cmd回车)，输入 netsh int ip reset 回车，再输入 netsh winsock reset，然后重启（如果你想快一点的话，shutdown -r -t 0，不用谢我）。

开梯子而出现的问题大部分应该都解决了。
