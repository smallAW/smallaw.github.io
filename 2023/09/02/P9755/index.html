题目传送门

废话不多说我们直接开始讲

思路

简单的循环模拟

由于第 $0$ 个小时细胞数（以下称 $sum$）为 $1$，我们从第 $1$个小时开始循环，随后按题意开始循环（$i$ 为循环变量，也就是当前时间）

  1. 如果 $i$ 为 $a$ 和 $b$ 的公倍数，那就结束当前循环（continue） ;

  2. 如果 $i$ 为 $a$ 的公倍数，也就是 $i\bmod a = 0$，那就把 $sum$ 翻倍（sum *= 2） ;

  3. 如果 $i$ 为 $b$ 的公倍数，也就是 $i\bmod b = 0$，那就把 $sum$ 除以 $2$ 再加向上取整（C++中整数除是直接把小数部分舍去，所以 sum = sum / 2 + 1），还要注意 $ans = 1$ 时， $ans \div 2$ 为 $0$，不能进行这个操作 ;

  4. 每次再把 $sum$ 对 $998244353$ 取模（sum %= 998244353） .

然后你就非常高兴的拿到了 90 分。

认真观察题目下面的提示，我们可以发现：

$\dfrac{a}{2} \equiv a \times 499122177 \pmod {998244353}$，即在对 $998244353$ 取模的情况下，您可以用 $a \times 499122177$ 代替 $a \div 2$

懂了吧就是在当 $i$ 为 $b$ 的公倍数情况下的操作是 sum *= 499122177 再进行取模操作。
